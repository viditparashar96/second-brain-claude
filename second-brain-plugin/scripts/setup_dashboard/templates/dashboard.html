Connect GitHub

Create a fine-grained Personal Access Token with these permissions:

  1. Go to GitHub Token Settings
  2. Click Generate new token
  3. Select repositories you want to track
  4. Permissions: Contents: Read, Pull Requests: Read/Write, Issues: Read/Write
  5. Copy the token and paste below

Connect Asana

Create a Personal Access Token from Asana:

  1. Go to Asana Developer Console
  2. Under Personal Access Tokens, click Create new token
  3. Name it "Second Brain" and create
  4. Copy the token and paste below

Connect Google (Gmail + Calendar)

You need a Google Cloud OAuth credential file. One-time setup:

  1. Go to Google Cloud Console
  2. Create a project (or select existing)
  3. Enable Gmail API and Google Calendar API
  4. Go to Credentials → Create Credentials → OAuth client ID
  5. Application type: Desktop app
  6. Download the JSON file
  7. Place it at:
    ~/.second-brain/data/credentials/gmail/credentials.json
  8. Come back here and click Retry
