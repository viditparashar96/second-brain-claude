# HEARTBEAT — Monitoring Checklist

> The heartbeat script reads this file to know what to check every 30 minutes.
> Edit this to add/remove monitoring targets.

## Gmail

- [ ] Unreplied emails older than 4 hours from clients or team
- [ ] Emails flagged as important/starred
- [ ] New emails from VIP contacts (team leads, direct reports, key clients)

## GitHub

- [ ] Open PRs awaiting my review
- [ ] PRs I authored with new comments or requested changes
- [ ] New issues assigned to me or my team
- [ ] CI failures on PRs I'm involved in

## Asana

- [ ] Tasks overdue (due date passed, not completed)
- [ ] Tasks due today
- [ ] Tasks due this week (upcoming 7 days)
- [ ] Blocked tasks needing attention

## Slack

- [ ] Unread messages in monitored channels
- [ ] DMs from team members that mention me or need a response
- [ ] Threads I'm participating in with new replies

## Habits

- [ ] Check HABITS.md for unchecked pillars
- [ ] Auto-detect objective achievements (PR merged, article saved)
- [ ] After 17:00 IST: nudge for any unchecked pillars
